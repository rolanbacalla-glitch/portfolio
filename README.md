# UX/UI Designer Portfolio

A modern, responsive portfolio website for UX/UI designers.

## 🚀 Quick Start

1. Open `index.html` in your browser to preview
2. Customize the content with your information
3. Add your project images to the `images/` folder
4. Deploy to Netlify, Vercel, or GitHub Pages

## 📝 Customization Guide

### Update Personal Information
- Edit your name in the hero section
- Update contact information (email, phone, location)
- Change bio and about text
- Add your social media links

### Change Colors
Open `css/style.css` and modify the CSS variables in `:root`:
```css
:root {
    --primary-color: #6366f1;    /* Change to your brand color */
    --secondary-color: #8b5cf6;  /* Accent color */
    --text-primary: #1f2937;     /* Main text color */
    --text-secondary: #6b7280;   /* Secondary text color */
}
```

### Add Project Images
1. Add your images to the `images/` folder
2. Update the `.project-image` background in each project card:
```html
<div class="project-image" style="background-image: url('images/RevolutiOS0.png');">
```

### Update Projects
Edit the project cards in `index.html`:
- Change project titles
- Update descriptions
- Modify tags
- Add case study links

## 🚀 Deployment

### Deploy to Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `portfolio` folder
3. Your site is live!

### Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import your project
3. Deploy with one click

### Deploy to GitHub Pages
1. Create a GitHub repository
2. Upload your files
3. Enable GitHub Pages in settings
4. Your site will be at `username.github.io/repository-name`
